jQuery Preview Get the Code »

jQuery Preview is a plugin that allows users to preview URLs before submitting them to be saved by an application. Facebook, Google+ and Yammer are all examples of tools with this functionality.

Try it out.

Enter your own url or select attach.

To get started you can view a number of different examples of ways to style your selector.

Status
Allows a user to type in a status and the URL is attached to the message.
Link
A simple url input that lets the user select a thumbnail.
Large Link
A Larger version of the link example with bigger thumbnails.
Rich Link
Also users to preview the video or photo before posting.
Multiple Inputs
Shows how to use jQuery Preview on Multiple inputs on a give page.
